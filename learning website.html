While Loop:


A while loop is another type of control flow statement in programming that allows a block of code to be executed repeatedly as long as a specified condition is true. The loop will continue executing as long as the condition remains true, and it will exit when the condition becomes false.

Do-While Loop:

        
A do-while loop is a variation of the while loop. The primary difference is that in a do-while loop, the block of code is executed at least once before the loop condition is checked. After the first execution, the loop continues to execute as long as the specified condition is true.

Arrays:

       
An array is a data structure in JavaScript that allows you to store multiple values in a single variable. Each value in an array is called an element, and each element has a unique index that starts from 0. Arrays are versatile and commonly used for organizing and manipulating sets of related data.
